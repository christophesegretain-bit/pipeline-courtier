⚙️ Configuration synchronisation Google Sheets
Pour activer la synchronisation complète (lecture + écriture)
Il faut créer un Google Apps Script en 3 minutes. Suivez le guide ci-dessous.
Étape 1 — Créer le Apps Script
  1. Allez sur script.google.com → Nouveau projet
  2. Collez ce code, remplacez l'ID du Sheet :
function doPost(e) { var ss = SpreadsheetApp.openById( '1Bfg8f2I-UQKuAv5u-AnQGSyIMv5iMUkwg5KkSdHGsQs' ); var sh = ss.getSheets()[0]; var body = e.postData.contents; try { var json = JSON.parse(body); if (json.type === 'presence') { var pres = {}; try { pres = JSON.parse(sh.getRange('B1').getValue()||'{}'); } catch(x){} pres[json.user] = json.ts; sh.getRange('B1').setValue(JSON.stringify(pres)); } else { sh.getRange('A1').setValue(body); } } catch(x) { sh.getRange('A1').setValue(body); } return ContentService.createTextOutput('OK'); } function doGet(e) { var ss = SpreadsheetApp.openById( '1Bfg8f2I-UQKuAv5u-AnQGSyIMv5iMUkwg5KkSdHGsQs' ); var sh = ss.getSheets()[0]; if (e.parameter.action === 'presence') { return ContentService.createTextOutput( sh.getRange('B1').getValue() || '{}' ); } return ContentService.createTextOutput(sh.getRange('A1').getValue()); }
Cliquez sur le code pour le sélectionner, copiez-collez dans Apps Script
Étape 2 — Déployer
  1. Cliquez Déployer → Nouveau déploiement
  2. Type : Application Web
  3. Accès : Tout le monde
  4. Cliquez Déployer → copiez l'URL
Étape 3 — Coller l'URL ici
En attendant — vos données sont sauvegardées localement sur cet appareil et lisibles depuis Google Sheets. La synchronisation entre appareils sera active dès que vous aurez configuré l'URL.
